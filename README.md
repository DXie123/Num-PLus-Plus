# Num++

A scientific & intutive calculator written in Flutter.

## Overview

`Num++` let you type math expression intuitively and evaluate it automatically. It supports basically all kinds of math function you will need. I'm an amateur coder, so feel free to give me any kinds of suggestions or open an issue. ðŸ¥°
